Random Number Visualization with Scatter and Line Plots¶

This Python code generates a sequence of 10 random integers and displays them using both a scatter plot and a line plot. The matplotlib library is used for plotting, while the random library generates the random numbers.

  • Data Generation: A list of 10 random integers between 1 and 100 is created for the y-values, with indices from 0 to 9 as the x-values.
  • Plotting Setup:
    • A figure is created with two subplots arranged side-by-side.
    • The left subplot displays a scatter plot, where each random number is represented by a blue dot.
    • The right subplot displays a line plot, where each random number is connected by a green line with markers at each point.
  • Customization:
    • Each plot includes titles, x-axis labels (Index), and y-axis labels (Random Value) for clarity.
  • Display: The tight_layout function optimizes the layout before displaying the plots side by side.

This code provides a simple example of using scatter and line plots to visualize data.

In [52]:
import matplotlib.pyplot as plt
from random import randint

# Generate a sequence of 10 random integers
y = [randint(1, 100) for _ in range(10)]
x = list(range(len(y)))

# Create a figure with two subplots
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(10, 5))

# Scatter plot
ax1.scatter(x, y, color='blue')
ax1.set_title("Scatter Plot of Random Numbers")
ax1.set_xlabel("Index")
ax1.set_ylabel("Random Value")

# Line plot
ax2.plot(x, y, color='green')
ax2.set_title("Line Plot of Random Numbers")
ax2.set_xlabel("Index")
ax2.set_ylabel("Random Value")

# Show the plots
plt.tight_layout()
plt.show()

import matplotlib.pyplot as plt
from random import randint

# Generate a sequence of 10 random integers
y = [randint(1, 100) for _ in range(10)]
x = list(range(len(y)))

# Create a figure with two subplots
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(10, 5))

# Scatter plot
ax1.scatter(x, y, color='purple', marker='s', s=100, edgecolor='black', alpha=0.7)
ax1.set_title("Scatter Plot of Random Numbers")
ax1.set_xlabel("Index")
ax1.set_ylabel("Random Value")

# Line plot
ax2.plot(x, y, color='orange', marker='D', markersize=8, linestyle='--', linewidth=2, alpha=0.8)
ax2.set_title("Line Plot of Random Numbers")
ax2.set_xlabel("Index")
ax2.set_ylabel("Random Value")

# Show the plots
plt.tight_layout()
plt.show()

sizes = [10, 20, 50, 100, 1000, 5000]
insertion_sort_times = [0.0010252999999999998, 0.0027026999999999954, 0.010147200000000002, 0.0381137, 3.6303399, 91.2180796]
merge_sort_times = [0.00161889999999687, 0.003635600000009731, 0.0090655000000055, 0.020108000000007564, 0.2687474000000094, 1.6147050999999948]


# Plot the data
